# Das Dashboard verstehen

Das Dashboard ist deine Startseite und zeigt dir auf einen Blick alles Wichtige.

## Bereiche auf dem Dashboard

- **Neueste Nachrichten** – aktuelle Beiträge der Schule und Lehrkräfte.
- **Offene Rückmeldungen** – Abfragen, auf die du noch antworten musst.
- **Termine** – anstehende Veranstaltungen und Schließtage.
- **Schnellaktionen** – Krankmeldung, Schickzeit ändern, Nachricht schreiben.

## Symbole und Farben

| Symbol | Bedeutung |
|--------|-----------|
|  Rot | Wichtig / dringend |
|  Gelb | Erinnerung / offen |
|  Grün | Erledigt / aktiv |
|  Blau | Information |

## Tipps

- Klicke auf jede Kachel, um zum Detailbereich zu springen.
- Ungelesene Nachrichten sind **fett** dargestellt.
- Über die **Glocke** oben rechts erreichst du alle Benachrichtigungen.
